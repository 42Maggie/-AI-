# 幻想世界AI创作项目：艾伦大陆知识库

从零用AI+人工微调构建的幻想世界，扩展世界观、角色叙事与内容模板。灵感源自《权力的游戏》+《阿凡达》。

## 项目目标
- 系统扩展世界框架（地理、文化）。
- 深化虚拟角色背景与互动语料。
- 优化AI提示工程模板（prompt engineering）。

## 仓库结构
- **world-framework.md**: 基础设定（地理、种族、宗教、经济）。
- **docs/**: 扩展文档（如生态细节）。
- **prompts/**: AI生成模板。
- **output/**: 最终内容输出（故事、互动文本）。
- **changelog.md**: 版本日志。

## AI创作流程
1. Prompt设计 → AI初稿（ChatGPT/Claude）。
2. 人工润色（确保自洽、宏大调性）。
3. Git版本管理 + 团队协作模拟（e.g., 飞书分享）。
